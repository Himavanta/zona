

字的命名要求 只能使用字母和数字，第一位必须是字母
字面量
  将常量压栈
  数字 
    无特殊符号 自动处理
    数字类型先不进行细致讨论，有可能后续会通过数字的后缀来区分精度和类型，比如 3u8
    123 1.3 -5
  布尔值
    0 为假，1 为真
  字符串
    单引号包裹 
      'hello'
    转义字符 \  
      '\n'
      'hello \'tom\''
    字符串模板 $
      实现较复杂，且规范不明确，先不实现
      'hello $' tom
    在运行时压入字符串地址和长度
定义字
  @ ;
  显式中途返回 $
循环
  回到字头部继续执行使用 ~
  在字的代码块中使用【? ~】或者【? 123 ! ~】来实现循环
  需要显式使用条件判断为有意设计
判断
  ? 和 ! 只读取接下来的一个字或者符号，所以代码块需要定义成字来被调用
  ? 或者 ? !
内存
  读使用 &
  写使用 #
注释
  行注释 _
  没有块注释
基础运算符
  只包含这几种 + - * / % ^
  没有 += -= *= /= ++ -- 等等
比较运算符
  只包含这三种 > < =
  没有 >= <= != == 等等

全部使用单符号，没有组合复合符号语法

还有待选的符号
  ( ) [ ] { } , . | " ` :
