# Zona — 设计决策记录

> 全部已定。性能优先。QBE 编译器。

---

## 1. 字符串：C 字符串 + 后置长度

`'hello'` → `p l`（指针 + 编译时长度）。Data section null 结尾（FFI 兼容），栈上独立长度（`:type` 不依赖 `\0`）。

**废案**：前置长度（传给 C 需偏移）、单指针引用模型（不透明，调试不便）、纯双值无 `\0`（FFI 边界需 `memcpy`）。

---

## 2. 内存读写：`&` `#` 直接操作

固定 8 字节。`&` 签名 `p:l|d`，`#` 签名 `lp:|dp:`。编译器 `loadl`/`loadd`/`storel`/`stored`。

**废案**：引用模型（`{addr, value}` — 解释器堆分配，调试不透明）、扁平原语 `:peek8 :peek32`（签名已区分类型）。

---

## 3. 编译器：QBE + 虚栈 + 寄存器传参

`@ fib d:d` → `function d $zona_fib(d %p0)`。函数内虚栈追踪，控制流边界保存/恢复，原语直接从虚栈取操作数。

**细节**：`?` 分支保存虚栈快照，false 分支恢复。`$` 返回 peek 虚栈顶。`~` 循环将虚栈刷到运行时栈。

**性能**：fib(35) 0.04s（C 1.0x）。

**废案**：ARM64 直接汇编（QBE 提供免费寄存器分配）、字节码 VM（性能天花板低）、解释器。（解释器已从项目中移除，仅编译器模式。）

---

## 4. 符号表

| 符号 | 语义 | | 符号 | 语义 |
|------|------|-|------|------|
| `@` `;` | 字定义 | | `+ - * / % ^` | 算术（类型化） |
| `:` | 签名分隔/原语前缀 | | `> < =` | 比较（返回 l） |
| `?` `!` `$` `~` | 控制流 | | `&` `#` | 内存读写 |
| `.` | 打印/成员访问 | | `_` `'` | 注释/字符串 |

---

## 5. 浮点常量

QBE 要求 `d_X.0` 格式。`%g` 输出可产生科学记数法，需注意极端值。

---

## 6. 开发顺序

✅ 编译器核心 + 类型 + 算术/比较/控制流
