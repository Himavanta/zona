## 已讨论不需要的

- 尾递归优化 — `~` 就是循环，不需要
- 块注释 — `_` 行注释够用
- 列号报错 — 行号已足够
- 具名导入导出 — 字典全局扁平
- HTTP / 异步 — 复杂度过高

## 设计备忘

- 宿主语言：C
- 解释器：`cc src/zona.c -o zona -lm -lreadline`
- 编译器：`cc src/zonac.c -o zonac -lm`（依赖 QBE）
- 显式路径前缀（`./` `../`）为未来标准库/包管理路径保留空间
- `[ ]` `( )` 是最有价值的待选符号，预留给匿名代码块或错误处理
- 字可覆盖重定义，后定义遮盖先定义
